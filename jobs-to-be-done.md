# Wrkr — Jobs to Be Done

> **Living document** · Updated: 2026-05-17
> **Use when:** evaluating whether a feature is in scope, understanding the motivation behind user behavior, designing proposal or governance flows.
> **Format:** *When [situation] → I want to [motivation] → so I can [outcome]* + the core anxiety the job resolves.
> **Related:** [strategy.md](strategy.md) · [user-task-flows.md](user-task-flows.md)

---

## Actors

| Actor | Description |
|---|---|
| **Worker** | Individual employee with no existing formal representation or safe coordination channel |
| **Organizer** | Worker who initiates proposals, recruits peers, tracks thresholds — informal role, any verified worker can fill it |
| **People Leader** | HR or CPO seeking genuine organizational insight rather than performative survey data |
| **Org Executive** | CEO or founder interested in participatory culture, succession, or cooperative transition |

## Phase 1 — Coordination & Visibility

### Worker

**W-01 · Safe Signal**
When I'm dissatisfied but don't know if others feel the same → I want to express sentiment without exposure → so I can find out whether I'm alone or part of a majority.
*Anxiety: What if I'm the only one? What if someone figures out it was me?*

**W-02 · Conditional Commitment**
When I'd support a collective action but only if enough others would too → I want to make a pledge tied to a threshold → so I can participate without taking a unilateral risk.
*Anxiety: What if I commit and no one else does? I'll have exposed myself for nothing.*

**W-03 · Quorum Awareness**
When I've committed → I want to see aggregate threshold progress (without seeing who else committed) → so I can gauge whether the action is likely to happen.
*Anxiety: Is anyone else actually doing this?*

**W-04 · Company Verification**
When I want to join my company's workspace → I want to prove I work there without revealing my name or details → so I can trust the space is real and safe.
*Anxiety: What if management plants fake accounts? What if my verification data identifies me?*

**W-05 · Issue Framing**
When I want to raise a concern → I want structured templates that help me articulate it clearly → so I get meaningful engagement rather than confusion or dismissal.
*Anxiety: I'll phrase this wrong and no one will support it.*

### Organizer

**O-01 · Coalition Discovery**
When I sense broader dissatisfaction but can't safely ask around → I want to discover whether others share my concerns and are willing to act → so I can understand the actual landscape before taking visible steps.
*Anxiety: I might be organizing a movement of one.*

**O-02 · Threshold Design**
When I'm creating a proposal → I want to set a threshold that is both meaningful and achievable → so the action that follows carries real legitimacy.
*Anxiety: Too low and it means nothing. Too high and it never triggers.*

**O-03 · Cohort Building Without Exposure**
When I want to grow participation → I want to share the proposal without being identified as the initiator → so I can build a coalition without becoming a target.
*Anxiety: Recruiting feels like organizing. Organizing feels dangerous.*

## Phase 2 — Organizational Health Intelligence

### Worker

**W-06 · Pattern Recognition**
When I've experienced a recurring problem → I want to see whether platform data shows a systemic pattern or a one-off → so I can understand whether collective investment is warranted.
*Anxiety: Maybe I'm imagining it.*

**W-07 · Consent-Gated Org Visibility**
When my employer wants aggregate data → I want to participate in a collective decision about whether to grant access → so I remain in control of what the organization sees about us.
*Anxiety: Will this get used against us?*

### People Leader

**PL-01 · Systemic Signal**
When surveys aren't revealing the truth → I want worker-consented aggregate data showing actual collective patterns → so I can address root causes rather than symptoms.
*Anxiety: I know something is wrong but I can't prove it or pinpoint it.*

**PL-02 · Non-Adversarial Channel**
When I want honest worker feedback → I want a channel workers actually trust — not associated with management surveillance → so I get truthful signals, not performative ones.
*Anxiety: Everything I set up gets gamed because workers know I see it.*

**PL-03 · Intervention Prioritization**
When the data reveals multiple concerns → I want ranked or clustered issue views with trend data → so I can allocate attention to what's most urgent, not just loudest.
*Anxiety: I'll fix the visible thing and miss the real thing.*

## Phase 3 — Democratic Workplace Infrastructure

### Worker

**W-08 · Meaningful Participation**
When decisions are being made that affect me → I want a formal, legitimate channel to participate in that process — not just react after the fact → so my voice has real weight.
*Anxiety: They'll do what they were going to do anyway.*

**W-09 · Representative Voice**
When I can't engage deeply with every issue → I want to elect representatives who reflect my values → so I'm represented without being constantly present.
*Anxiety: I don't have time for this and I'll be steamrolled.*

**W-10 · Deliberation Before Decision**
When a major proposal is on the table → I want time to understand tradeoffs before any vote → so I make informed decisions rather than reactive ones.
*Anxiety: I'll vote for something I'll regret.*

### Org Executive

**OE-01 · Participatory Culture Infrastructure**
When I want genuine worker participation built into how we operate → I want a platform that provides structure, tools, and legitimacy mechanisms → so participation is embedded, not performative.
*Anxiety: "Open door" and "town halls" aren't working.*

**OE-02 · Decision Legitimacy**
When decisions need worker input → I want a documented process showing that threshold support was met → so decisions carry collective endorsement and can't be later contested.
*Anxiety: We'll make a decision and half the company will say they had no say.*

## Phase 4 — Cooperative Transition Infrastructure

### Worker

**W-11 · Ownership Pathway Clarity**
When my workplace is considering employee ownership → I want clear scenario-based models showing what different structures look like → so I can make an informed decision rather than a leap of faith.
*Anxiety: I don't understand what I'm actually voting on.*

**W-12 · Genuine Stewardship**
When my workplace becomes worker-owned → I want to participate meaningfully in governance — not just own shares passively → so I experience co-ownership rather than a financial formality.
*Anxiety: "Worker-owned" will just mean different bosses.*

### Org Executive

**OE-03 · Succession Without Sale**
When I'm thinking about exit → I want an alternative to external acquisition that preserves culture, mission, and people → so the company continues on its own terms.
*Anxiety: Selling means watching everything I built get dismantled.*

## Jobs We Are Not Solving

| Job | Why excluded |
|---|---|
| Anonymous venting / public reviews | No coordination mechanism; adversarial framing; no action pathway |
| Formal union organizing | Legally complex; adversarial framing undermines MVP adoption |
| Management performance reviews | Employer-facing; violates trust architecture |
| Individual grievance tracking | Creates individual exposure risk; platform is aggregate-only |
| Legal / labor dispute support | Out of scope; refer to legal partners |
| Public-facing company ratings | Becomes reputation management, not coordination infrastructure |
